# AiHomeDesign | AI智能家居定制设计

## 📖 项目背景

基于多年的室内设计行业经验，结合当前 AI 大语言模型的爆发式发展，本项目旨在探索利用 AI 技术自动化生成室内定制设计图纸的可行性与落地实践。

## 🛠️ 技术路线

初期曾尝试开发 CAD 插件直接调用 AI 进行自动化绘图，但综合考量技术难度、实现成本与稳定性后，本项目转向了更轻量、易拓展的 Python 脚本生成方案。

**核心实现路径：大语言模型 (LLM) + Python `ezdxf` 库**

1. **构建 AI Prompt**：设计专业的系统提示词（Prompt），引导 AI 准确理解家居定制设计需求与结构逻辑。
2. **代码生成**：AI 接收指令后，输出基于 Python `ezdxf` 库的绘图代码。
3. **图纸输出**：运行生成的 Python 代码，自动打包输出 `.dxf` 格式的标准 CAD 图纸。
4. **预览与深化**：生成的结果可直接导入 AutoCAD、浩辰CAD、CAD看图王等软件进行图纸查看与二次深化修改。

## 🎯 项目目标 —— AI 驱动的智能化全案设计

实现“一键全案设计”的终极目标任重道远，但本项目正通过分阶段的探索逐步逼近。

### 🟢 [50%] 阶段一：AI 柜体定制设计（当前阶段）

- **使用方法**：将项目内的 `AI_Home_Design_Prompt_Final.md` 作为系统预设喂给 AI 模型，随后输入你的具体设计需求。

- **Prompt 示例**：

  > “请根据文档设计餐边柜，尺寸是：宽度3080，高2550，深度你自由发挥。风格是现代轻奢，顶部是双眼皮吊顶，深300，右边靠墙。” *(注：以上仅为示例，实际使用时风格与细节可自由发挥，但**必须提供明确的硬性约束条件**，如具体尺寸、靠墙情况、梁柱关系等。)*

- **当前进展**：

  - ✅ 目前 `AI Prompt` 已完成，可实现 AI 对于衣柜/餐边柜的结构化定制设计。
  - ✅ 方案设计逻辑表现良好，能为设计师提供高效的草图参考与灵感起点。
  - ⏳ **为什么说只完成了 50%？** 目前的测试与优化主要基于 **Gemini 模型**，尚未在其他主流大模型（如 GPT-4、Claude 或国内大模型）上进行全面测试。因此，该 Prompt 的跨模型通用性仍无法完全保障。

- **开源共建**： 图纸在实际落地细节（如生产工艺、收口逻辑等）上仍有考究空间。未来会持续调整迭代 `AI Prompt`，非常欢迎各位设计师和开发者提出宝贵建议，参与共创。
